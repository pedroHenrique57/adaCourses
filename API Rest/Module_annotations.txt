Rest= https://pt.wikipedia.org/wiki/REST 
(Representational State Transfer) define um conjunto de restrições a serem usadas para a criação de um tipo especial de serviços-Web, denominados Web services RESTful, 
que fornecem interoperabilidade entre sistemas de computadores na Internet; RESTful permite que os sistemas solicitantes acessem e manipulem representações textuais de 
recursos da Web usando um conjunto uniforme e predefinido de operações sem estado (requisição e resposta independentes).
